This is a Vite + React + TypeScript project, styled with Tailwind CSS v4 and routed with React Router.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the home page by modifying `src/pages/home.tsx`. New pages go in `src/pages/`, get a route in `src/App.tsx`, and a link in `src/components/sidebar.tsx`.

Persistent data lives in `localStorage`, read/written through `src/lib/db.ts` (`readDb`/`writeDb`).
